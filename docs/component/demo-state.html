2. State & Array

複数要素(.class)の取得と状態(.state)の一括操作です。Select multiple elements and manipulate state.

📝 コード📝 Code

const states = ['active', 'info', 'disabled'];
let count = 0;

// .class で全要素にクリックイベントを紐付け
carbon.class('state-btn').tapped((widget) => {
    widget.state.clear(); // 既存の状態をリセット
    widget.state.apply(states[count % 3]);
    count++;
});
