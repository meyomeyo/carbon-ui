CarbonUI Documentation

CarbonUIは、「読んだらわかる」に特化したjQueryライクなモダンUIライブラリです。
シンプルなメソッドチェーンで、状態管理、ジェスチャーイベント、簡易ルーティングなどを直感的に記述できます。

CarbonUI is a modern, jQuery-like UI library aiming for "read and understand".
With a simple method chain, you can intuitively write state management, gesture events, and simple routing.

初期設定 (Init)Setup (Init)

ルーティング等のSPA機能を利用する場合は、carbon.init()を呼び出します。単体でDOM操作を使うだけなら不要です。

When using SPA features like routing, call carbon.init(). It is not necessary if you only use standalone DOM operations.

import carbon from './carbon-ui.js';

carbon.init({
    root: '#app',
    home: 'component/home',
    basePath: ''
});

DOM操作DOM Actions

id, class, find 等のメソッドで要素を取得し、チェーンメソッドで操作します。

Get elements with methods like id, class, and find, and manipulate them using chained methods.

carbon.id('my-btn').text('Click me!').show();
carbon.class('items').hide();
carbon.find('.container > .item').html('<span>Hello</span>');

状態管理 (State)State

state オブジェクトを使用して、carbon-* クラスや data-carbon-state を直感的に操作できます。

Use the state object to intuitively manipulate carbon-* classes and data-carbon-state.

const btn = carbon.id('btn');
btn.state.apply('active'); // "carbon-active" を追加 / Add "carbon-active" class
btn.state.is('active');    // true
btn.state.remove('active');
btn.state.clear();         // 全ての carbon-* クラスを削除 / Remove all carbon-* classes

イベント・ジェスチャーEvents & Gestures

スマートフォン等でも使いやすいジェスチャーイベントを標準サポートしています。

Native support for gesture events that are easy to use on smartphones.

  • tapped(fn): タップまたはクリックtapped(fn): Tap or click
  • longPressed(fn, ms): 長押しlongPressed(fn, ms): Long press
  • swiped(fn, threshold): スワイプ (方向を取得可能)swiped(fn, threshold): Swipe (direction available)
carbon.id('swipe-area').swiped((w, dir) => {
    console.log(`Swiped: ${dir}`); // 'left', 'right', 'up', 'down'
});

SPAルーティングSPA Routing

carbon.go()とcarbon.afterGo()を用いた軽量ルーターを内蔵しています。

Built-in lightweight router using carbon.go() and carbon.afterGo().

// 遷移後に毎回実行されるフック / Hook executed after every transition
carbon.afterGo((pageName) => {
    console.log(`${pageName} に遷移しました！ / Navigated to ${pageName}!`);
});

// 手動でページ遷移 / Manual page transition
carbon.go('component/about');
