ℹ️ Routing (SPA)

ここは fetch() によって component/about.html から動的に読み込まれたコンポーネントです。This component was dynamically loaded via fetch from component/about.html.

キャッシュ機構を利用しているため、2回目以降の遷移は瞬間的に切り替わります。Due to the caching mechanism, subsequent transitions are instantaneous.

📝 コード📝 Code

// 他のページから移動するとき
carbon.go('component/about');

// 戻るとき
carbon.back();
