CarbonUI Documentation

CarbonUIはシンプルで軽量なJavaScript UIライブラリです。jQueryライクなメソッドチェーンを用いながら、モダンな状態管理やSPAルーティング、各種ジェスチャーをサポートします。

CarbonUI is a simple and lightweight JavaScript UI library. It supports modern state management, SPA routing, and various gestures while using a jQuery-like method chain.

1. Core API (要素取得・ライフサイクル)1. Core API (Select & Lifecycle)

carbon.* オブジェクトから呼び出せる基本メソッドです。

Basic methods called from the carbon.* object.

要素の取得

Select Elements

MethodReturnsDescription
carbon.id(id)CarbonWidget指定したID(#不要)の要素を取得します。Get element by ID (no #).
carbon.class(className)CarbonEls指定したクラス(.不要)を持つ全ての要素を取得します。Get elements by class name (no .).

4. State (状態管理API)4. State API

状態管理APIです。carbon-[name]が付与されます。State management API. Automatically applies carbon-[name] class.

MethodDescription
.apply(name)状態を適用します。Applies state.
.remove(name)状態を解除します。Removes state.
.is(name)適用中か返します。Returns true if state is applied.
.current()現在の状態名を返します。Returns current state name.
.clear()すべての状態をクリア。Clears all states.
carbon.id('btn').state.apply('loading'); // `carbon-loading` クラスを要素に付与
console.log( carbon.id('btn').state.current() ); // "loading" を出力

6. Router (SPA)6. Router (SPA)

FetchとHistoryを用いた超軽量ルーター機能。Lightweight SPA router using Fetch and History APIs.

carbon.init(options)

ルーターを初期化します。Initialize SPA router.

carbon.init({
    root: '#app',          // 描画先DOM (デフォルト: '#app')
    home: 'component/home' // アクセス時に最初に読み込むパス (.htmlは不要)
});
carbon.go(pageName, replace = false)

ページ遷移を非同期で実行します。Navigate asynchronously via fetch().

carbon.back()

一つ前のページへ戻ります。Navigate back in history.
